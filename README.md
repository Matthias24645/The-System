# The-System
## 🔥 System App

A Solo Leveling-inspired training system with levels, quests, boss fights, and skill progression.

---

## 🚀 How to Use

1. Download the repository (ZIP)
2. Extract the files
3. Open `index.html` in your browser

---

## ⚠️ Important

* Keep all files in the same folder
* Do not move or rename sound files

---

## 🎯 Features

* Daily Quests
* Level & Rank System
* Boss Quests (from Level 10)
* Attributes & Skill Progression
* Level-Up Animations & Sounds

---

## 🔄 Updates & Community

This project is actively being developed.

💬 Feedback is welcome – I read suggestions and improve the system
🚀 Regular updates with new features (bosses, skills, animations, etc.)

## ⚔️ Vision

The goal is to turn this system into a full Solo Leveling-style web game.
